# Crud-Pedido-PRECIFY

♻️ Página de Compras de Material Reciclável
Este projeto é uma página web responsiva voltada para a compra de materiais recicláveis, desenvolvida com HTML, CSS, Bootstrap e JavaScript. O objetivo é facilitar a negociação entre catadores, empresas e demais interessados na compra de materiais recicláveis.

🚀 Tecnologias Utilizadas
HTML5 — Estrutura da página

CSS3 — Estilização customizada

Bootstrap 5 — Layout responsivo e componentes visuais

JavaScript (Vanilla JS) — Funcionalidades interativas

🧩 Funcionalidades
Listagem de materiais recicláveis disponíveis para compra

Botões interativos de compra e filtros

Design responsivo para mobile e desktop

Integração visual com Bootstrap

Código modular e organizado
